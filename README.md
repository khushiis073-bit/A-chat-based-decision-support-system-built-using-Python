# Chat-Based Decision Support System

This is a simple Python-based interactive decision support system.
It helps users make decisions by asking questions and providing
suggestions based on their responses.

## Features
- Interactive chat-style input
- Beginner-friendly Python logic
- Easy to extend with more rules

## How to Run
1. Download the file `decision_support.py`
2. Run the file using:
   python decision_support.py
